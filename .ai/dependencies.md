# Project Dependencies

Below are the primary libraries defined in `package.json`:

---

## 1. Production Dependencies

- **`next`** (16.2.10) & **`react`** (19.2.4): Core web framework.
- **`@neondatabase/auth`** (0.4.2-beta): Official Neon integration for credentials/OAuth sign-in.
- **`pg`** (8.22.0): Non-blocking PostgreSQL client to execute queries.
- **`axios`** (1.18.1): Promise-based HTTP client for API requests.
- **`@tanstack/react-query`** (5.101.2): Server state manager client.
- **`react-hook-form`** (7.81.0) & **`@hookform/resolvers`** (5.4.0): Form bindings and Zod integration.
- **`zod`** (4.4.3) & **`validator`** (13.15.35): Schemas and data sanitation.
- **`cloudinary`** (2.10.0): SDK to manage photo uploads.
- **`browser-image-compression`** (2.0.2): Compresses image files locally on client side.
- **`jszip`** (3.10.1) & **`file-saver`** (2.0.5): Bundles and saves selected photos into ZIP archives.
- **`tailwindcss`** (4) & **`@tailwindcss/postcss`** (4): Utility-first styling engine.
- **`clsx`** (2.1.1) & **`tailwind-merge`** (3.6.0): Combines Tailwind CSS classes.
- **`class-variance-authority`** (0.7.1): Manages UI component state styling.
- **`sonner`** (2.0.7): Lightweight, customizable notifications.
- **`lucide-react`** (1.23.0): Reusable icons.
- **`next-themes`** (0.4.6): Handles dark/light theme options.
- **`bcryptjs`** (3.0.3) & **`uuid`** (14.0.1): Cryptography and identifier helper.
- **`date-fns`** (4.4.0): Custom date transformations.

---

## 2. Dev Dependencies
- **`typescript`** (5): TypeScript support.
- **`eslint`** (9) & **`eslint-config-next`** (16.2.10): Linter rules.
- **`@types/node`** (20), **`@types/pg`** (8.20.0), **`@types/react`** (19), **`@types/react-dom`** (19): Types packages.
